# python

Repository for Confluence wiki migration and helper scripts.

This repository contains scripts to export and import Confluence pages, manage attachments, and handle Gliffy diagrams.

Usage

- See `src/wiki_export_and_import.py` for the main migration tool.

Notes

- `wiki_down_upload_export/` is excluded from version control by default (it's large). If you want to include it, remove it from `.gitignore` and consider using Git LFS for large files.
